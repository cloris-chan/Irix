# Irix preview packages

Irix is a work-in-progress UI and rendering framework experiment. These NuGet packages are currently unlisted previews used to reserve package IDs and verify package consumption from the repository's own Windows Poc.

The packages are not a stable public SDK. APIs, package layout, runtime behavior, diagnostics, and supported platforms may change without compatibility guarantees before any listed release.

Repository: https://github.com/cloris-chan/Irix
